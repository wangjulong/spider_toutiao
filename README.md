# spider_toutiao
# 作用：python 爬取头条号图集（装修效果图）
### 使用说明：
1. pip install requests 
2. python spider.py

### 参数设置
1. KEYWORDS：想要搜索的关键词
2. HOME_DIR：需要把文件保存到什么位置
3. GROUP_START：开始的索引（相对于头条网站的第几批加载的数据，可以分批处理，或断点续传）
4. GROUP_END：结束的索引
